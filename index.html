What is JavaScript ?

Javascript is one of the most popular programing languages, it is considered as the language of the Web , it makes the web pages alive.If we make a comparison between the human body and the web pages , HTML will be like the skelton (bones) , CSS will be like the skin and JavaScript will be like the muscles which moving everything. There are at least three great things make JavaScript unique , first Full integration with HTML/CSS. second , Simple things are done simply and third, Support by all major browsers and enabled by default.

What is the deference between JavaScript and ECMAScript ?

ECMAScript :
The full form of ECMA is European Computer Manufacturer's Association. ECMAScript is a Standard for scripting languages such as JavaScript, JScript, etc. It is a trademark scripting language specification. JavaScript is a language based on ECMAScript. A standard for scripting languages like JavaScript, JScript is ECMAScript. JavaScript is considered as one of the most popular implementations of ECMAScript.
JavaScript :
JavaScript is an implementation of the ECMAScript standard.

What is the deference between JavaScript and Java ?

JavaScript :
JavaScript is a lightweight programming language(“scripting language”) and used to make web pages interactive. It can insert dynamic text into HTML. JavaScript is also known as browser’s language. JavaScript(JS) is not similar or related to Java. Both the languages have a C like a syntax and are widely used in client-side Web applications, but there are few similarities only.
Java :
Java is an object-oriented programming language and have virtual machine platform that allows you to create compiled programs that run on nearly every platform. Java promised, “Write Once, Run Anywhere”.
